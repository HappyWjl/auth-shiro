## 前言

`auth-shiro`项目致力于打造一个完整的，智能的，简洁的权限系统，采用现阶段流行技术实现。

## 项目介绍

`auth-shiro`项目是一套搜索系统，包括 **登录权限、接口权限、菜单权限**。
采用JAVA语言，并结合redis、MySQL、Postman实现数据搜索、同步数据。
